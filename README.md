# Waifu Generator

A customizable waifu generator with safe & NSFW options, image downloads, and trait control.

## Features
- Create your own waifu
- SFW + NSFW modes
- Download waifus as PNG, JPEG, or WebP
- API fallback option

## How to Use
- Open `index.html` in browser.
or
- Open the website.
